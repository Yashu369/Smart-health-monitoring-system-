# Smart Health Monitoring System

This project is an IoT-based health monitoring system using ESP32.

Sensors Used:
- MAX30102 (Heart Rate & SpO2)
- MLX90614 (Temperature)

The system sends data to Firebase Realtime Database.

A web dashboard displays patient health data.

If abnormal health conditions occur, alert notifications are sent to caregivers.

Technologies Used:
- ESP32
- Firebase
- HTML
- CSS
- JavaScript
- Machine Learning
